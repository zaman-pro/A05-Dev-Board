Question-1: What are the different ways to select an element in the DOM?

There are several ways to select an element in the DOM.

  • 01. getElementById(): Selects an element by its unique id
  • 02. getElementsByClassName(): Selects elements by their class
  • 03. getElementsByTagName(): Selects elements by their tag name
  • 04. querySelector(): Selects the first element that matches the CSS selector
  • 05. querySelectorAll(): Selects all elements that match the CSS selector

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The difference between innerHTML, innerText, and textContent is:

  • innerHTML: It represents the HTML content inside an element (including HTML tags). It can be used to get or set HTML content.
  • innerText: It represents the visible text content of an element, also HTML tags. It can be affected by CSS styles like: (display:none)
  • textContent: Represents the text content of an element, as well as all nested elements, but does not account for CSS styling.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where you add a single event listener to a parent element instead of adding one to each child element. The parent listens for events that bubble up from the child elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process where an event triggered on a child element "bubbles" up to its parent elements. In other words, when an event happens on a nested element, it is first handled by that element, and then it moves upwards through its parent elements.

Question-5: How do you create, add, and remove elements using JavaScript?

  • Create an element: We use document.createElement() to create a new element.
  • Add an element: We use methods like appendChild(), prepend(), or insertBefore() to add elements to the DOM.
  • Remove an element: We use removeChild() to remove a child element from a parent, or remove() to remove an element directly.

This is how we create, add, and remove elements using JavaScript.
